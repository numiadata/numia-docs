---
title: Getting Started
---

To begin using Numia's Advanced API and access rich blockchain data, follow the steps below to create an account, set up a project, and generate your API key.

1. First, you need to create an account with Numia. Visit our dashboard and [sign up](https://app.numia.xyz/login) to get started. 
2. Once logged in, click **Create** on the **New Project** card, and choose a name for your project
    
    <Image 
    src="/media/api/create-project.gif"
    />
    
3. After creating your project, you need to open it and add the blockchain protocol you want to access through the API. 
    
    <Image 
    src="/media/api/add-protocol.gif"
    />
4. Get your API key to be used for Bearer Token authentication in your API requests, allowing you to interact with Numiaâ€™s data. You'll also be able to see the LCD and RPC endpoints for that chain. 
5. Feel free to check the endpoints documentation and send request using the API docs playground.
