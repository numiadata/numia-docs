The following example presents how you can you request an ad via API and its response.

## Request
Use the following cURL command to request an ad:

```bash
curl -X GET "https://api.staging.numia.xyz/campaign/:address?position_id=:position_id" \
  -H "Accept: application/json" \
  -H "Authorization: Bearer :bearer_token"
```

| **Parameter**      | **Type** | **Description**                                                                 |
|--------------------|---------------------|---------------------------------------------------------------------------------|
| `address`          | Path Parameter      | Chain address of the connected wallet.                                          |
| `global`           | Path Parameter      | If the ad is not wallet-aware, the publisher should pass `address=global`.      |
| `position_id`      | Query Parameter     | Unique identifier for the ad placement to be filled (e.g., `wallet_home`).      |
